Como usar os métodos da API

Buscar um sócio por E-mail e Data de Nascimento

Exemplo de envio usando o método $.ajax do jQuery

                            
// O Token de acesso é único e é necessário para realizar consultas na API
$.ajax({
  method: 'POST',
  url: 'https://pagseguro.spr.org.br/socio/api/v2/integracao.php',
  headers: { Token: 'tsfTRQoLDhuMubdAS8wAWkY5vUEsODuI' },
  data: {
      method: 'getByDTNascEmail',
      dtnascimento: '1970-01-01',
      email: 'ejsalgado@globo.com'
  }
}).done(function(res){
  console.log(res)
}, function(err){
  console.log(err.responseJSON)
})
